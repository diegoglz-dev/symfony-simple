# Symfony-simple

Es un mini proyecto con las bases del framework Symfony. El proyecto est√° creado para ir instalando paquetes a medida que se van precisando.

## Para levantar el proyecto

Se debe tener instalado:

-   PHP
-   Composer
-   MySql
-   Node y NPM

Ejecutar el comando

`symfony serve`

### Instalando sistema de plantillas Twig

En la terminal se debe ejecutar el siguiente comando:

`composer require symfony/twig-pack`

### Instalando herramienta debug

En la terminal se debe ejecutar el siguiente comando:

`composer require symfony/debug-pack`
